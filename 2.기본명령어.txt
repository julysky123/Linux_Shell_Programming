

<ls>
-> 디렉토리의 내용을 출력하는 명령어
-> -l: 현재 디렉터토리의 내용을 자세하게 출력. 
-> -a: 숨김파일도 출력. (파일이름 앞에 .)
-> 옵션은 두개이상 동시에 사용해도 되고 순서를 바꿔도 된다. (ex: -la)
-> ls 디렉토리명 : 틀정 티렉토리의 내용 확인.
-l 파일명 : 해당 파일의 정보가 출력.
-l 하위디렉토리 : 하위디렉토리 아래의 파일들을 자세히 출력.
-ld 하위디렉토리 : 하위디렉토리의 정보가 자세히 출력.
-F 디렉토리인 경우 이름의 끝에 /를 추가.
-r 출력 결과를 역순으로 표시.
-R 하위디렉토리를 순회하면서 그 안의 내용을 출력.
-t 파일의 수정 시간을 오름차순으로 정렬하여 출력.

<cd>
 -> . 현재 경로
 -> .. 상위 경로
 -> cd 만 적으면 최상위 경로로
 -> cd~ 최상위 경로
 -> ./는 생략이 가능.
 -> cd - 이전작업 디렉토리로 이동.

-참고-
ls 명령어도 절대 경로와 상대 경로 모두 사용할 수 있다.
.. 이용.

<cat>
  -> concatenate의 약자. 두개의 파일을 붙이는 명령어
  -> cat 파일명 
  !파일명은 반드시 텍스트 파일이어야한다
  -n : 줄 번호와 같이 출력
  -b : 비어 있지 않은 줄만 출력

<more>
  -> 텍스트 파일을 출력하는 명령어
  -> cat과 달리 화면 제어가 된다.
  -> 명령키 
    1.space : 다음 페이지    2. b : 이전페이지
    3.enter : 한줄 이동      4. q : 종료.
   
<less>
  -> more 명령어를 개선한 명령어.
  -> space, pg dn, f : 다음 페이지
  -> b, pg up : 이전 페이지
  -> 위쪽 방향키, k : 한줄 위
  -> 아래 방향키 , j : 한줄 아래
  -> G: 문서의 끝으로 이동
  -> g: 문서의 처음으로 이동
  -> q: 종료
  -> /문자열 : 아래 방향으로 이동하면서 입력된 문자열을 찾는다.
  
  
<head / tail>
  -> 파일의 처음이나 끝 부분을 확인할 떄 사용하는 명령어
  -> head 파일명.
  -> default : 10줄 출력
  -n 3 파일명: 3줄 출력.
  
  -> tail의 옵션중 -f를 사용하면 파일에 대한 변화를 실시간으로 확인할 수 있다.
